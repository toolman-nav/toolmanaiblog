> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring the Astro static site in line with `toolmanai-seo-fix-spec-v2.md`.

**Architecture:** Keep Astro as a fully static site and move SEO-critical URL generation into shared route helpers and content metadata. Preserve Chinese display names while using explicit ASCII slugs for all public URLs, redirects, sitemap, RSS, and cross-links.

**Tech Stack:** Astro 7.0.5, Astro content collections, zod schemas, @astrojs/sitemap, @astrojs/rss, Node-based verification scripts.
